# KZPushSegue

This is a custom NSStoryboardSegue to make a nice iOS-like push transition. 

## Support

Swift 4.0+

## Objective-C version

if you use Objective-C,you can use [GRPushSegue](https://github.com/insidegui/GRPushSegue).
